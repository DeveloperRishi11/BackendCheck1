# Task Management API

A simple REST API built with Node.js and Express for managing tasks.

## Features
- Create, read, update, delete tasks
- Filter tasks by status and priority  
- Get task statistics
- RESTful design with proper HTTP status codes

## Quick Start
```bash
npm install
npm run dev
